# movies
 Movie RESTFull API Service

# system requirement
  * Maven 3 
  * Java 17 
  * mysql
   
# How to deploy
  * create database on mysql named "movies"
  * clone this repo 
  * go inside "movies" directory -> run "cd movies"
  * run "mvn clean install"
  * run "mvn spring-boot:run"
